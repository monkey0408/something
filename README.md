## 📋 使用/查看说明
1.  **下载方式**：
    - 直接下载仓库 ZIP 包，或用 Git 克隆整个仓库。
    - 务必**完整下载所有零件文件 + 装配体文件**，否则打开装配体时会提示“找不到零件”。
2.  **打开步骤**：
    1.  先打开所有 `.sldprt` 零件文件（或直接打开 `.sldasm` 装配体，让软件自动关联零件）。
    2.  若出现“零件缺失”提示，检查零件文件是否和装配体文件在同一目录下。
3.  **修改建议**：
    - 如需修改模型，建议先另存为副本再编辑，避免覆盖原始文件。
    - 关键尺寸/参数已标注在工程图中，可根据实际需求调整。
